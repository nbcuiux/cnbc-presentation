Requirements(initial)

  • Text EditorOpen or Close

    Goal

    Have a simple and intuitive User Interface to write and edit the news articles.

    Current features

    Content and Presentation are separate

    Create paragraph blocks and embed media items

    Must-haves

    Simple, intuitive, and minimal UI

    Quick and seamless experience, reactive, responsive

    Contextual options: Different options presented when block level elements are selected vs. inline elements like text

    Potentially geared for collaboration, if implemented correctly

    Other

    Likes Medium Editor’s separation of content from presentation

  • Wire StoriesOpen or Close
    Wire Story

    Goal

    Ingest external sources and publish as an article

    Current features

    Automatically ingested

    Rules to determine the source and assigns the Franchise

    Editor has the ability to edit part of the article if needed

    Other

    c.f. USA uses ingesting from 3rd source as well with The Verge and Mr. Robot collaboration

  • Breaking NewsOpen or Close
    Homepage

    Goal

    Quickly publish out time-sensitive news in the prominent part of the website

    Current features

    Breaking News headline appears in a red bar below the navigation menu

    Headline is a clickable link to the article

    Must-haves

    Quickly publish out the news

    Other

    Editor in the Hot Seat position would publish out the breaking news

  • OtherOpen or Close

    Search

    Currently, there are two types of search - saving search, which is the often search keywords of the user, and editorial search

    Wants Elasticsearch: display the related content to the keyword

    Workflow

    To consider different workflows for editors and writers

    Communication tool

    Currently communicating and tracking the status with Google Sheets

    Wants tools like Slack integrated into CMS

    Promoting Content to Homepage

    Currently, you can mark a content as promoted (megaphone icon), which gets displayed under the corresponding Franchise on the Homepage

    Other nice-to-haves

    Social integration into CMS: currently nothing in CMS. What goes into CNBC official account is managed by the social team, and writers will individually post their own article. Interesting feature to consider.

    Statistics for each article, how well it’s doing (nice to have) - currently done in Omniture
